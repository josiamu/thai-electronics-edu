# ชนิดType ย่านความถี่Frequency Range การใช้งานApplication

3 Function Generator
(ฟังก์ชันเจเนอเรเตอร์)Function Generator		 0.01 Hz – 10 MHz+ ห้องปฏิบัติการ กำเนิดรูปคลื่นหลายแบบLab use; generates multiple waveform types

(สวีปเจเนอเรเตอร์)Sweep Generator		 สแกนช่วงความถี่Frequency sweep range ทดสอบตัวกรองความถี่ (Filter)Filter and frequency response testing

Function Generator มี 7 ส่วนสำคัญ ในวงจรภายใน

A Function Generator has 7 key internal circuit sections

# ชื่อส่วนประกอบComponent Name หน้าที่Function
1 วงจรปรับความถี่Frequency Control Network กำหนดความถี่ของสัญญาณที่ต้องการSets the desired output frequency
2 แหล่งจ่ายกระแสคงที่ (Constant Current Source)Constant Current Source (×2) จ่ายกระแสคงที่ให้วงจรอินติเกรเตอร์ทำงานได้Provides stable current for integrator operation
3 วงจรอินติเกรเตอร์ (Integrator)Integrator Circuit กำเนิดคลื่นสามเหลี่ยม (Triangle wave)Generates triangle wave
4 วงจรมัลติไวเบรเตอร์ (Multi-vibrator)Multi-vibrator Circuit กำเนิดคลื่นสี่เหลี่ยม (Square wave)Generates square wave
5 วงจรขยายเอาต์พุต (Output Amplifier)Output Amplifier (×2) ปรับขนาดแอมพลิจูด 0.1VP-P ถึง 10VP-PAdjusts amplitude from 0.1VP-P to 10VP-P
6 สวิตช์เลือกสัญญาณ (Selector Switch)Selector Switch เลือกชนิดรูปคลื่นที่ต้องการส่งออกSelects the desired output waveform type
7 วงจรเปลี่ยนรูปคลื่น (Shaping Circuit)Shaping Circuit แปลงคลื่นสามเหลี่ยมเป็นคลื่นไซน์ โดยใช้ R + DiodeConverts triangle to sine wave using R + Diode network
คุณสมบัติที่ดีของ Function Generator Characteristics of a Good Function Generator
  • ปรับความถี่ได้กว้าง 1 Hz – 1 MHz ค่าความถี่คงที่และแม่นยำ อ่านค่าได้ชัดเจน
  • สัญญาณ ไม่ผิดเพี้ยน (low distortion) ไม่มีสัญญาณรบกวน (low noise)
  • ปรับ Amplitude ได้ต่อเนื่องตั้งแต่ต่ำสุดถึงสูงสุด
  • เลือกรูปคลื่นได้หลายแบบ: Sine, Triangle, Sawtooth, Square, Pulse
  • มีความต้านทานเอาต์พุต 50 Ω มาตรฐาน ลดการสูญเสียกำลัง
  • Wide frequency range 1 Hz – 1 MHz with stable, accurate, readable frequency
  • Signal has low distortion and low noise
  • Continuously adjustable Amplitude from minimum to maximum
  • Multiple waveform selections: Sine, Triangle, Sawtooth, Square, Pulse
  • Standard 50 Ω output impedance to minimize power loss

ตัวอย่างที่ 7-1 — Sine Wave 20VP-P, 1 kHz

Example 7-1 — Sine Wave 20VP-P, 1 kHz

โจทย์: ตั้งค่า Sine wave, 20VP-P, f = 1 kHz
Task: Set Sine wave, 20VP-P, f = 1 kHz
  1. กดสวิตช์เปิดเครื่อง (ปุ่ม 1)
  2. กดปุ่มเลือกคลื่นไซน์ Sine (ปุ่ม 3)
  3. กด Ampl (ปุ่ม 8) แล้วหมุนปุ่ม Knob จนจอแสดง 20.00 VP-P
  4. กด Freq (ปุ่ม 7) แล้วหมุนปุ่ม Knob จนจอแสดง 1.00 kHz
  5. ตรวจสอบด้วย Oscilloscope ที่ขั้ว Output (ปุ่ม 11)
  1. Press Power switch (Button 1)
  2. Press Sine wave button (Button 3)
  3. Press Ampl (Button 8) and turn Knob until display shows 20.00 VP-P
  4. Press Freq (Button 7) and turn Knob until display shows 1.00 kHz
  5. Verify with Oscilloscope at Output terminal (Button 11)

ตัวอย่างที่ 7-2 — Square Wave 2VP-P, 100 Hz

Example 7-2 — Square Wave 2VP-P, 100 Hz

โจทย์: ตั้งค่า Square wave, 2VP-P, f = 100 Hz
Task: Set Square wave, 2VP-P, f = 100 Hz
  1. กดสวิตช์เปิดเครื่อง (ปุ่ม 1)
  2. กดปุ่มเลือกคลื่นสี่เหลี่ยม Square (ปุ่ม 4)
  3. กด Ampl (ปุ่ม 8) แล้วหมุนปุ่ม Knob จนจอแสดง 2.00 VP-P
  4. กด Freq (ปุ่ม 7) แล้วหมุนปุ่ม Knob จนจอแสดง 100 Hz
  5. ตรวจสอบด้วย Oscilloscope ที่ขั้ว Output (ปุ่ม 11)
  1. Press Power switch (Button 1)
  2. Press Square wave button (Button 4)
  3. Press Ampl (Button 8) and turn Knob until display shows 2.00 VP-P
  4. Press Freq (Button 7) and turn Knob until display shows 100 Hz
  5. Verify with Oscilloscope at Output terminal (Button 11)
ข้อควรระวัง:
• ตรวจสอบให้แน่ใจว่าแรงดัน Output ที่ตั้งไม่เกินค่าทนทานของวงจรที่ทดสอบ
• เชื่อมต่อสายดิน (GND) ก่อนต่อสัญญาณ Output เสมอ
• ใช้สายโคแอกเชียล (Coaxial Cable) ต่อระหว่าง Function Generator กับ Oscilloscope
Cautions:
• Ensure the set output voltage does not exceed the test circuit's tolerance
• Always connect GND before connecting the output signal
• Use coaxial cable between Function Generator and Oscilloscope
