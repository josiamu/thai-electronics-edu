ตัวต้านทาน (Resistor)

รหัสสี 4 แถบ / 5 แถบ • EIA96 • วงจรอนุกรม/ขนาน

หน้าที่และชนิดของตัวต้านทาน
  • จำกัดปริมาณกระแสไฟฟ้าในวงจร
  • แบ่งแรงดันไฟฟ้า (Voltage Divider)
  • ปรับค่าสัญญาณในวงจร
  • หน่วย: โอห์ม (Ω)
ชนิดของตัวต้านทาน
• Fixed Resistor — ค่าคงที่
• Variable Resistor (VR) — ปรับค่าได้
• Thermistor (NTC/PTC) — เปลี่ยนตามอุณหภูมิ
• LDR — เปลี่ยนตามแสง
• Varistor (VDR) — เปลี่ยนตามแรงดัน

วิธีอ่าน 4 แถบ: ค่า = (แถบ1 × 10 + แถบ2) × ตัวคูณ ± ค่าผิดพลาด
ตัวอย่าง: น้ำตาล-ดำ-แดง-ทอง = (1×10+0)×100 = 1,000 Ω = 1 kΩ ±5%
รหัสสี 5 แถบ (5-Band Color Code)

แถบที่ 1-3 = ตัวเลข 3 หลัก, แถบที่ 4 = ตัวคูณ, แถบที่ 5 = ค่าผิดพลาด

วิธีอ่าน 5 แถบ: ค่า = (แถบ1×100 + แถบ2×10 + แถบ3) × ตัวคูณ ± ค่าผิดพลาด
ตัวอย่าง: แดง-แดง-แดง-น้ำตาล-น้ำตาล = (222)×10 = 2,220 Ω = 2.22 kΩ ±1%
วงจรตัวต้านทาน

วงจรอนุกรม (Series)

RT = R₁ + R₂ + R₃ + ...
  • กระแสเท่ากันทุกตัว: IT = I₁ = I₂ = I₃
  • แรงดันรวม: ET = E₁ + E₂ + E₃
  • ความต้านทานรวมมากกว่าทุกตัว

วงจรขนาน (Parallel)

1/RT = 1/R₁ + 1/R₂ + 1/R₃ + ...
  • แรงดันเท่ากันทุกตัว: ET = E₁ = E₂ = E₃
  • กระแสรวม: IT = I₁ + I₂ + I₃
  • ความต้านทานรวมน้อยกว่าทุกตัว
สูตรพิเศษ 2 ตัว: RT = (R₁ × R₂) / (R₁ + R₂)
