สูตรทั้ง 3 วงจร Formulas for All 3 Circuits
อนุกรมSeries
RT = R₁+R₂+R₃
I = V/RT (เท่ากันทุก R)
ขนานParallel
1/RT = 1/R₁+1/R₂+1/R₃
V เท่ากันทุก branch
ผสม (R1 อนุกรม กับ R2‖R3)Mixed (R1 series + R2‖R3)
Rp = R₂R₃/(R₂+R₃)
RT = R₁+Rp
คำแนะนำการสังเกต Observation Guide
  • ความเร็วจุด ∝ ขนาดกระแสในเส้นทางนั้น
  • สีตัวต้านทาน ฟ้า→แดง ตามกำลังไฟฟ้า (ร้อน)
  • อนุกรม: I เท่ากัน, V แยกตาม R
  • ขนาน: V เท่ากัน, I แยกตาม R — R น้อย = I มาก (จุดเร็วกว่า)
  • ผสม: R1 ตก V ส่วนหนึ่ง, ที่เหลือแบ่งใน R2‖R3
  • เพิ่ม R ใดก็ตาม → RT เพิ่ม → IT ลด
  • Dot speed ∝ current magnitude in that path
  • Resistor color blue→red by power dissipation (heat)
  • Series: same I, voltage splits by R
  • Parallel: same V, current splits by R — lower R = more I (faster dots)
  • Mixed: R1 drops some voltage; remainder shared across R2‖R3
  • Increase any R → RT increases → IT decreases
