Inductor & Transformer

ตัวเหนี่ยวนำ (Inductor / Coil)Inductor / Coil
  • เก็บพลังงานในรูปสนามแม่เหล็ก
  • ต่อต้านการเปลี่ยนแปลงกระแส
  • ใช้ใน Filter, Oscillator, Power Supply
  • หน่วย: เฮนรี (Henry, H)
  • Stores energy in a magnetic field
  • Opposes changes in current
  • Used in Filters, Oscillators, Power Supplies
  • Unit: Henry (H)

หม้อแปลงไฟฟ้า (Transformer)Transformer
  • แปลงแรงดัน AC ขึ้นหรือลงตามอัตราส่วนรอบขด
  • ขดปฐมภูมิ (Primary) รับแรงดันเข้า
  • ขดทุติยภูมิ (Secondary) ส่งแรงดันออก
  • ไม่เปลี่ยนความถี่
  • กำลังไฟฟ้าเข้า ≈ กำลังไฟฟ้าออก (ไม่คิดสูญเสีย)
  • Steps AC voltage up or down according to the turns ratio
  • Primary winding receives input voltage
  • Secondary winding delivers output voltage
  • Does not change frequency
  • Input power ≈ output power (ignoring losses)
V₁/V₂ = N₁/N₂ = I₂/I₁
ชนิดของหม้อแปลง
• Step-Up: แรงดันออก > แรงดันเข้า (N₂ > N₁)
• Step-Down: แรงดันออก < แรงดันเข้า (N₂ < N₁)
• Isolation: แรงดันเท่ากัน แยก Ground
• Auto Transformer: ขดเดียว มี Tap

ตัวอย่าง:
หม้อแปลง 220V/12V มีรอบขด N₁:N₂ = 220:12
ถ้า I₂ = 1A → I₁ = 12/220 = 0.055 A Types of Transformers
• Step-Up: Output > Input voltage (N₂ > N₁)
• Step-Down: Output < Input voltage (N₂ < N₁)
• Isolation: Same voltage, separate ground
• Auto Transformer: Single winding with tap

Example:
Transformer 220V/12V has turns ratio N₁:N₂ = 220:12
If I₂ = 1A → I₁ = 12/220 = 0.055 A
